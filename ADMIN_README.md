# Инструкция по использованию админки

## Установка и настройка

1. Выполните миграции:
```bash
php artisan migrate
```

2. Заполните базу данных начальными данными:
```bash
php artisan db:seed
```

## Доступ к админке

- **URL**: `/admin/login`
- **Email**: `admin@admin.com`
- **Пароль**: `password`

⚠️ **Важно**: После первого входа измените пароль администратора!

## Функционал админки

### 1. Управление пунктами меню (`/admin/menu-items`)

- Просмотр всех пунктов меню
- Добавление новых пунктов
- Редактирование существующих пунктов (нажмите "Редактировать")
- Удаление пунктов
- Управление порядком отображения (поле "Порядок")
- Включение/отключение пунктов (чекбокс "Активен")

**Поля:**
- **Название** - текст, отображаемый в меню
- **URL** - ссылка пункта меню (можно оставить `#`)
- **Порядок** - порядок отображения (меньше = выше)
- **Активен** - показывать/скрывать пункт на сайте

### 2. Управление преимуществами (`/admin/advantages`)

- Просмотр всех преимуществ
- Добавление новых преимуществ
- Редактирование существующих преимуществ
- Удаление преимуществ
- Управление порядком отображения

**Поля:**
- **Название** - заголовок преимущества
- **Описание** - дополнительное описание (опционально)
- **Иконка** - путь к иконке (например, `/assets/block.svg`)
- **Порядок** - порядок отображения
- **Активен** - показывать/скрывать на сайте

**Примечание**: Преимущества отображаются в блоке "Товары, которые покупают чаще всего" на главной странице.

## Структура данных

### Пункты меню (menu_items)
- Отображаются в шапке сайта
- Максимальное количество не ограничено
- Отображаются только активные пункты

### Преимущества (advantages)
- Отображаются в секции "О нас" на главной странице
- Рекомендуется создать 6 преимуществ для полного заполнения всех блоков
- Отображаются только активные преимущества

## Безопасность

- Все маршруты админки защищены middleware `auth`
- Неавторизованные пользователи перенаправляются на страницу входа
- Авторизованные пользователи не могут попасть на страницу входа (редирект на `/admin`)

## Изменение пароля администратора

Вы можете изменить пароль через tinker:

```bash
php artisan tinker
```

Затем выполните:
```php
$user = App\Models\User::where('email', 'admin@admin.com')->first();
$user->password = bcrypt('новый_пароль');
$user->save();
```
